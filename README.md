# Employee Tax Optimization System

## Description
A MERN-based application that distributes total company salary among employees in an optimal way to minimize total income tax, based on given tax slab rules.

## Features
- Accepts number of employees and total yearly salary
- Distributes salary equally to minimize tax
- Calculates employee-wise tax and take-home salary
- Implements slab-wise tax and additional surcharge rules

## Tech Stack
- React.js
- Node.js
- Express.js
- Plain CSS

## How to Run
1. Start backend:
   - cd backend
   - npm install
   - npm start

2. Start frontend:
   - cd frontend
   - npm install
   - npm start
